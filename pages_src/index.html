About

LiquidLava is a JavaScript MVC framework for building user interfaces, which scale from simple interactive elements on page to complex feature-rich applications. What makes it different from other frameworks is transparent internal design: this means, that you always know, what you are doing, and what happens inside the framework.

Features

  • Unique class system: generated classes are almost as fast as their hand-written analogues
  • Templates are parsed into configs for JavaScript classes (not compiled into functions)
  • Never creates DOM elements directly. Renders them as HTML and inserts into DOM
  • Software patterns instead of dependency injection
  • No magic!
