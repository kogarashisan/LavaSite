Standard widgets showcase

These widgets are included with framework by default. Note: they are not limited to bootstrap theme - you can write your own CSS and even HTML markup for any widget.

Customizing the code is very easy: you are always encouraged to inherit your own classes, write your own templates and use these widgets as examples when creating your own.

Note: framework has also a collection of form input widgets, which are not shown here, like tri-state Checkbox or Radio.

All headers and all content of almost any widget in Lava is not just a static text, but a normal Lava template, so inside you can use other widgets and data binding. Also, most widgets support styling via "resources", so you can apply styles without changing the markup.
