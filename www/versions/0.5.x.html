0.5.1

Fixes

  • Fixed bug with requestAnimationFrame in Opera and Safari.

0.5.0

Feature:

  • Sugar of CollapsiblePanel, Collapsible and Accordion widgets now has "root_resource_name" (stores all attributes from sugar tag to widget's container).
  • Added Lava.system.FocusManager, currently with very basic capabilities.
  • Improved Lava.bootstrap() - now it processes control attributes on elements with "lava-app" attribute.
  • Added "label" role to input.Abstract - assigns the "for" attribute.

Refactoring

  • Tree widget: part of "node_body" template was moved to "node_body_content" template.
  • Lava.widget.Standard#inject now has default position - "Top"
  • Sugar now adds attributes to container from "root_resource_name", instead of replacing them.
  • Removed `//depends:{}` operator from ExpressionParser.

Fixes

  • Fixed Lava.system.DataView.
  • Fixed bug in event handling, which crashed application when focused element was removed from DOM.
  • Fixed bug in scope refresh system, which crashed the application.
  • Fixed bug in view refresh system which could lead to browser hangup.
  • Fixed a parsing issue in IE (IE produces duplicate attributes on some elements).
  • Fixed Lava.scope.Binding in "from_widget" mode.

Build also contains minor fixes, refactorings and improvements.
