Feature

  • New way to create refreshers in Foreach and If views: createRefresher() instance method, that can be used during roles registration.

Refactoring

  • Removed x:broadcast directive and associated functionality - it's redundant (same can be done with roles).
  • Serializer is now a separate class - Lava.system.Serializer. Default instance can be accessed as `lava.serializer`.
  • Changed protected API in Foreach and If views.
  • edit_template widget definition tag is now marked as ALPHA.
  • Tree widget: animation is now disabled by default, but can be turned on via config option (you can assign refresher for nodes)

Fix

  • Now x:equiv attribute is correctly removed from parsed tag.
