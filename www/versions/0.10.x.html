0.10.1

Refactoring

  • Removed Lava.scheduleRefresh(). Modified widget.Standard#inject() and injectIntoExistingElement().
  • ScopeManager.lock() and unlock() now count locks, so it can be locked multiple times.

Fix

  • Tree widget: template now uses "is_expandable" dynamic scope rather than direct reference.
  • Fixed a potential bug with translation of plural strings.
  • Fixed bug in event dispatch algorithm: under certain conditions a call to ViewManager#cancelBubble could cancel the next bubbling event.
  • Fixed a bug in ViewManager: crash, when global role target was removed inside event handler.
  • Refreshers: improved performance and sorting behaviour. Added refresher.Standard#broadcastInDOM(). Modified broadcastRemove() method in views and refresher.
  • Fixed 2 bugs in ExpressionParser:
    - now it correctly recognizes the "{&gt;" sequence (escaped expression opening)
    - does not report a space between array (" [") as error.

Site

  • Fixed a couple of bugs on documentation page.

0.10.0

Once again reworked the Refresher API.

Upgrade:

  • In templates: if there are Emulated containers with refresher, then you will need to rename "appender" option to "prepender" (if you need - then you will see exceptions in console).
  • "insertion_strategy" and "removal_strategy" options were removed - remove them from templates.

Feature

  • Added _cRefresher#get_start_element_callback and _cRefresher#get_end_element_callback.
  • Added Firestorm.DOM#moveRegionBefore, Firestorm.DOM#moveRegionAfter.

Refactoring

  • Refactored refreshers: removed *_strategies fields, default strategies became more clever.

Fix

  • Refreshers now correctly sort templates.
