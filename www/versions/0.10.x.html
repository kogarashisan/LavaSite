0.10.x

Once again reworked the Refresher API.

Upgrade:

  • In templates: if there are Emulated containers with refresher, then you will need to rename "appender" option to "prepender" (if you need - then you will see exceptions in console).
  • "insertion_strategy" and "removal_strategy" options were removed - remove them from templates.

Feature

  • Added _cRefresher#get_start_element_callback and _cRefresher#get_end_element_callback.
  • Added Firestorm.DOM#moveRegionBefore, Firestorm.DOM#moveRegionAfter.

Refactoring

  • Refactored refreshers: removed *_strategies fields, default strategies became more clever.

Fix

  • Refreshers now correctly sort templates.
