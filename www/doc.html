Welcome to LiquidLava documentation

LiquidLava is an MVC framework with unique architecture, designed for rapid development of complex web applications.

Firestorm is a low-level library, which does not modify prototypes of native JavaScript objects, which makes it extremely reusable and simple.

Documentation is divided into the following sections:

  • Tutorials - will help you to make your first steps with the framework
  • Reference - Detailed explanation of every aspect of the framework
  • API - Generated automatically from source code files
